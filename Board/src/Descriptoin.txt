Задача.
1.  Есть шахматная доска.
    В классе определяются два массива фигур - armyBlack armyWhite;
    Доска - класс общих полей и методов работы с шахматными фигурами.
    В классе происходит инициализация всех фигур. Инициализация -
    первоначальное определение свойств фигур.

    Также класс содержит информацию о состоянии всех полей шахматной доски.
    - цвет поля
    - фигура (или null)
    - Place признак enPassant (адрес пешки которую можно взять на проходе)




    - метод boolean isEmptySquare (PLace place) возвращает true, если
    поле place свободно и false если занято

    - метод boolean isOpponentSquare (boolean color, Place place) возвращает
    true если поле place занято фигурой цвета !color

    - метод Place enPassant () возвращает не null в том случае,
    если в текущий момент доступно взятие на проходе пешки, прошедшей
    положение place

2.  Есть два набора по 16 шахматных фигур

    Каждая фигура имеет свойства:
    - цвет
    - жизнь
    - положение
    - область передвижения
    - пешка имеет дополнительное поле

    Метод фигуры возвращает все возможные положения для следующего хода.
    При этом он использует методы доски, проверяющие доступность поля.
    boolean isEmptySquare - свободное поле
    boolean isOpponentSquare - поле, занятое фигурой оппонента, которую
    можно взять.
    Пешки также отдельно проверяют возможность взятия на проходе
    Place enPassant - поле, только что пройденное пешкой соперника
